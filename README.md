# Malicious Activity Detection on Social Media

ML + NLP pipeline to detect **spam**, **fake accounts**, and **hate speech** in social media content.

## Features

| Detector | Algorithm | Macro F1 |
|---|---|---|
| Spam Detection | Random Forest + TF-IDF + text stats | 1.00 |
| Fake Account Detection | Gradient Boosting + account features | 1.00 |
| Hate Speech Classification | Calibrated LinearSVC, 3-class | 0.96 |

- **NLP pipeline**: URL/mention cleaning → tokenisation → stopword removal → Porter stemming
- **Feature engineering**: TF-IDF (bigrams, 5000 features) + 12 hand-crafted text statistics
- **OCR support**: extract text from uploaded images via pytesseract (optional)
- **Web interface**: Flask app with dark-themed UI for real-time analysis

## Setup

```bash
pip install -r requirements.txt

# Optional: install Tesseract for OCR
brew install tesseract   # macOS

# Train models (auto-runs on first app launch too)
python train.py

# Start web app
python app.py
# → http://localhost:5050
```

## Project Structure

```
├── src/
│   ├── preprocessor.py      # Text cleaning and normalisation
│   ├── feature_engineer.py  # TF-IDF + text/account feature extraction
│   ├── models.py            # SpamDetector, FakeAccountDetector, HateSpeechDetector
│   ├── evaluator.py         # Metrics, confusion matrices, ROC curves
│   └── ocr_extractor.py     # pytesseract wrapper with graceful fallback
├── data/
│   └── generate_data.py     # Synthetic dataset generation
├── templates/
│   └── index.html           # Flask web UI
├── models/                  # Saved .pkl model files (generated by train.py)
├── plots/                   # Evaluation plots (generated by train.py)
├── train.py                 # Train + evaluate all models
└── app.py                   # Flask app (port 5050)
```

## API

**POST /analyse**
```json
{ "text": "FREE CLICK NOW http://bit.ly/win", "image_b64": null }
```
Response:
```json
{
  "spam":         { "label": "Spam",    "flagged": true,  "probability": 0.97 },
  "fake_account": { "label": "Real",    "flagged": false, "probability": 0.12 },
  "hate_speech":  { "label": "Neutral", "flagged": false, "probability": 0.88,
                    "class_probs": {"Neutral": 0.88, "Offensive": 0.09, "Hate": 0.03} },
  "overall_flagged": true
}
```
